JavaScript

JavaScript adalah bahasa pemrograman yang digunakan untuk membangun situs web dari sisi klien. JavaScript membuat situs web lebih dinamis dan interaktif. Sesuatu yang tidak dapat Anda lakukan dengan HTML atau CSS. Bahasa pemrograman JavaScript adalah bahasa penulisan tingkat tinggi(dekat bahasa manusia) yang berjenis scripting (scripting language) yang ditulis secara dinamis (dynamically typed) dan dieksekusi oleh seorang Interpreter. JS memiliki konvensi penulisan yang mirip dengan bahasa manusia. Ini membuat JavaScript lebih mudah. Untuk pemula sekalipun. Sebuah bahasa scripting berarti bahwa JavaScript adalah bahasa pemrograman tertanam dalam program yang lebih besar seperti web browser. Jadi, ini adalah program yang dapat menggunakan JavaScript. Pengetikan dinamis berarti bahwa variabel dalam program yang dikodekan dalam JavaScript dapat berisi tipe data yang berbeda. yang dimana dapat memasukkan angka dalam variabel dan kemudian diubah menjadi huruf tanpa kesalahan. JavaScript harus dijalankan oleh Interpreter. Ini berarti bahwa itu harus diterjemahkan ke dalam bahasa yang dimengerti komputer agar dapat berjalan. Sekarang, proses penerjemahan ini dilakukan secara otomatis oleh browser web Anda.

Sejarah JavaScript

JavaScript adalah salah satu bahasa tertua di dunia. Itu telah ada dengan nama aslinya Mocca sejak 1995. Dia kemudian beralih ke LiveScript sebelum menstandarisasi EcmaScript hingga akhirnya mendapatkan nama JavaScript. JavaScript pertama kali dikenalkan oleh Netscape pada tahun 1995. Pada awalnya JavaScript hanya digunakan untuk membuat website dinamis. Namun seiring berjalannya waktu, JavaScript mulai digunakan untuk membuat aplikasi mobile, membuat game, dan lain sebagainya. EcmaScript sendiri merupakan bahasa scripting standar yang dikembangkan oleh European Computer Manufacturers Association (ECMA). EcmaScript memungkinkan JavaScript untuk berjalan dengan lancar di berbagai browser. Dengan sejarahnya yang panjang, tidak heran JavaScript adalah bahasa pemrograman paling populer, menurut survei Stack Overflow. Secara keseluruhan, 97,7% situs web di Internet menggunakan bahasa pemrograman JavaScript.

Kelebihan JavaScript

JavaScript memiliki banyak kelebihan, diantaranya:

  • JavaScript dapat digunakan untuk membuat website dinamis
  • Open Source, Gratis dan dapat dikembangkan luas
  • Tidak Memerlukan Instalasi lain, cukup menggunakan text editor dan browser
  • Ekosistem dan Komunitas yang Luas
  • Multi platform, bisa dijalankan di banyak sistem operasi

Penulisan JavaScript

JavaScript dapat ditulis di dalam HTML atau di dalam file terpisah dengan ekstensi .js. Untuk menuliskan JavaScript di dalam HTML, dapat menggunakan tag script atau sedangkan untuk file ekstensi terpisah menambahkan atribut src pada tag "" untuk menunjukkan lokasi file JavaScript.
